# {{site.data.keyword.watson}} SDK
{: #using-sdks}

SDK 讓與應用程式開發相關聯的大部分複雜性變得抽象化。透過您所知的語言提供程式設計介面，可以協助您快速開始使用 {{site.data.keyword.ibmwatson_notm}} 服務。
{: shortdesc}
