# {{site.data.keyword.watson}} SDK
{: #using-sdks}

SDK 将与应用程序开发相关联的大部分复杂性抽象化。通过以您已经了解的语言提供编程接口，这些 SDK 可以帮助您快速入门和熟悉运用 {{site.data.keyword.ibmwatson_notm}} 服务。
{: shortdesc}
