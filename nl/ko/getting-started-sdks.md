# {{site.data.keyword.watson}} SDK
{: #using-sdks}

SDK에는 애플리케이션 개발과 연관된 대부분의 복잡한 사항들이 요약되어 있습니다. 이는 사용자가 익숙한 언어로 프로그래밍 인터페이스를 제공함으로써 {{site.data.keyword.ibmwatson_notm}} 서비스를 빠르게 작성하고 실행할 수 있도록 도와줍니다.
{: shortdesc}
